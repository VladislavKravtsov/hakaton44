# hakaton44
# Ore Grade Classifier — классификация сортности руды по фото аншлифов
Очень важно!!! Github не дал возможность загрузить модель в репозиторий из-за ограничения размера файла! Прошу установить модель с яндекс диска: https://disk.yandex.ru/d/GnmO1LzEF2ueWA
Веб-сервис, который по изображению аншлифа определяет **сортность руды** (рядовая / труднообогатимая / оталькованная), строит маску фаз, считает метрики и формирует отчёт.

## Возможности

- Анализ одного фото: класс, маска фаз, наложение (overlay), числовые метрики.
- Выгрузка результата в **PDF** и **CSV**.
- **Пакетный режим**: загрузка папки снимков, прогресс, скачивание ZIP (маски + overlay + `statistics.csv`).
- **Дообучение с экспертом**: правка масок кистью в интерфейсе и fine-tune модели на своих данных.
- Работа с гигапиксельными панорамами на GPU с 6 ГБ VRAM (тайловый инференс).

## Как работает

1. **Предобработка** — CLAHE (LAB) + шумоподавление.
2. **Сегментация на 4 класса** (фон, обычные срастания, тонкие срастания, тальк): нейросеть DeepLabV3+ (`segmentation-models-pytorch`) тайлами 512×512 с перекрытием 64 px; при отсутствии модели — резервный CV-алгоритм (Оцу + морфология).
3. **Метрики** — доли фаз + морфология зёрен (фрагментация, доля мелких зёрен, медианная площадь).
4. **Классификация сортности** — дерево решений (`scikit-learn`) поверх признаков + жёсткое правило по проценту талька.

## Требования

- Python 3.10+ (проверено на 3.12)
- Windows / Linux
- GPU опционально (для CUDA поставить torch с нужным индексом; по умолчанию ставится CPU-версия)

## Установка и запуск

### Windows (быстро)

Дважды кликните `run.bat` — он создаст виртуальное окружение, поставит зависимости и поднимет сервер. Лог пишется в `run_log.txt`.

### Вручную

```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/Mac: source .venv/bin/activate
pip install -r requirements.txt
# PyTorch (пример для CUDA 12.1; для CPU просто: pip install torch torchvision)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121

uvicorn main:app --host 127.0.0.1 --port 8000
```

Откройте **http://127.0.0.1:8000**

> Для установки нужно ~3–4 ГБ свободного места (в основном PyTorch).

## API

| Метод | Endpoint | Назначение |
|---|---|---|
| GET  | `/` | Веб-интерфейс |
| GET  | `/api/health` | Проверка работоспособности |
| POST | `/api/analyze` | Анализ фото → метрики + маска + overlay (base64) |
| POST | `/api/analyze/pdf` | Анализ → PDF-отчёт |
| POST | `/api/analyze/csv` | Анализ → строка метрик CSV |
| POST | `/api/batch` | Пакетная обработка папки (возвращает `job_id`) |
| GET  | `/api/batch/{job_id}/status` | Прогресс задачи |
| GET  | `/api/batch/{job_id}/download` | Скачать ZIP с результатами |
| POST | `/api/recompute` | Пересчёт метрик по исправленной маске |
| POST | `/api/corrections` | Сохранить экспертную правку как обучающий пример |
| POST | `/api/retrain` | Запустить дообучение модели |
| GET  | `/api/retrain/status` | Статус дообучения |

## Структура проекта

```
webapp/
├── main.py                  # FastAPI-приложение и API
├── inference.py             # сегментация + метрики + классификация сортности
├── report.py                # генерация PDF-отчётов
├── train.py                 # обучение сегментационной модели
├── fit_grade_classifier.py  # обучение классификатора сортности
├── evaluate_grades.py        # оценка качества
├── build_own_dataset.py     # подготовка своего датасета
├── import_lumenstone_output.py  # импорт внешнего датасета
├── talc_from_contour.py     # разметка талька
├── static/index.html        # веб-интерфейс
├── checkpoints/             # веса модели (best_model.pt, grade_tree.joblib)
├── requirements.txt
└── run.bat                  # запуск в один клик (Windows)
```

## Обучение

```bash
# сегментация
python train.py --data-root own_dataset_prepared --arch deeplabv3plus --epochs 30

# классификатор сортности (по признакам из масок)
python fit_grade_classifier.py
```

## Классы и цвета маски

| Класс | Значение | Цвет |
|---|---|---|
| 0 | фон / вмещающая порода | чёрный |
| 1 | обычные срастания | зелёный |
| 2 | тонкие срастания | красный |
| 3 | тальк | синий |
